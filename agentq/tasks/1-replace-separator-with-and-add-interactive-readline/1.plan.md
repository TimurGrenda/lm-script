# Task 1: Modify parse_args() to use `:` instead of `--`

**Plan**: 1-replace-separator-with-and-add-interactive-readline
**Status**: Pending

## Goal

Replace the `--` prompt separator with `:` in the argument parser, and add support for interactive readline mode when `:` is the last argument.

## Context

The `parse_args()` function in `lm` (lines 65-99) currently uses `--` as the separator between llm options and the prompt text. The `--` separator causes bash syntax errors when the prompt contains shell metacharacters like `(`, `)`, backticks, etc., because bash interprets them before `lm` runs.

The function sets two globals: `LLM_OPTS` (array of flags for llm) and `PROMPT` (the prompt text). We need to add a third: `INTERACTIVE_PROMPT` (boolean flag for readline mode).

Current code at line 74-79:
```bash
case "$1" in
  --)
    shift
    # Everything remaining after "--" is the prompt
    PROMPT="$*"
    break
    ;;
```

## Implementation

**File**: `lm` (lines 65-99)
**Action**: modify

1. Update the comment block (lines 65-67) from:
   ```
   # Parse arguments: everything before "--" is passed to llm as options,
   # everything after "--" is the prompt text. This avoids maintaining a catalog
   # of every llm flag — new flags pass through automatically.
   ```
   to:
   ```
   # Parse arguments: everything before ":" is passed to llm as options,
   # everything after ":" is the prompt text. If ":" is the last argument
   # (nothing after it), interactive readline mode is triggered.
   # Using ":" instead of "--" avoids bash syntax errors when the prompt
   # contains shell metacharacters like parentheses or backticks.
   ```

2. Initialize `INTERACTIVE_PROMPT=false` alongside the other globals (after line 71).

3. Replace the `--` case (lines 74-79) with:
   ```bash
   :)
     shift
     if [[ $# -eq 0 ]]; then
       # No text after ":" — trigger interactive readline mode
       INTERACTIVE_PROMPT=true
     else
       # Everything remaining after ":" is the prompt
       PROMPT="$*"
     fi
     break
     ;;
   ```

## Verification

- Source the script and call `parse_args : hello world` → `PROMPT` should be "hello world", `INTERACTIVE_PROMPT` should be "false"
- Call `parse_args :` → `PROMPT` should be "", `INTERACTIVE_PROMPT` should be "true"
- Call `parse_args -m gpt-4 : hello` → `LLM_OPTS` should contain "-m gpt-4 -t general", `PROMPT` should be "hello"
- Run `shellcheck lm` — must pass
